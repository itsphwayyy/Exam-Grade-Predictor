# Exam-Grade-Predictor
This is a lightweight app built with python ans streamlit.
# How this work?
CAT - 40% 
  - 8% for submitting homework regularly
  - 8% for finishing classwork
  - 8% for sumbitting projects
  - 15% for your behavior in the classroom
    

Last Exam Mark - 10%

- 10% for how many days you study per week
- 10% for hours of revise a day
- 10% for enough sleep
- 10% for eating healthy food
- 10% enough resting

# You need to rate yourself out of 10 except for your last exam mark, study day per week, and hour of revise a day.
